Python 合并微信与支付宝账单
========================

这份示例文档归档了用 Python、pandas 合并微信与支付宝账单并生成 Markdown 财务报告的核心流程。

## 数据导入

- 支付宝账单通常需要跳过说明行后读取 CSV。
- 微信账单需要统一交易时间、交易类型、收支、金额、备注等列名。
- 金额字段要去除货币符号、空格和千分位分隔符后再转为数值。

## 清洗规则

- 退款、交易关闭、0 元订单不计入有效支出。
- 账户之间转账不重复计入收入或支出。
- 同一订单号、同一时间和同一金额的重复记录需要合并或剔除。

## 输出报告

最终报告按收入、支出、净现金流、分类支出和月度趋势生成 Markdown 表格，便于归档到个人知识库。
