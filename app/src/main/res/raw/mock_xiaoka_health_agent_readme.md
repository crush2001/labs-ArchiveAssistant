小卡健康 Agent 仓库
=================

xiaoka-health-agent 是一个本地文件驱动的 AI 健康助手项目，使用规则、模板、知识库和脚本组织健康管理流程。

## 仓库结构

- agent.md：Agent 角色、目标和行为约束。
- SOUL.md：长期偏好、家庭成员背景和健康关注点。
- SKILL.md：可复用技能说明。
- config：模型、路径和运行参数。
- templates：体检报告、用药记录、饮食日志等模板。
- references：医学指标解释和健康管理资料。
- fixtures：测试输入与输出样例。
- scripts：导入、清洗和归档脚本。

## 归档价值

该文档适合作为工程实践类文档 mock，也能覆盖健康档案与本地 Agent 主题。
