health-coach 开源健康教练
======================

health-coach 是一个面向 AI 健康教练的知识库示例，围绕营养、体检指标、训练计划、补剂和用药信息组织 Markdown 文档。

## 知识模块

- nutrition：饮食照片、热量估算和宏量营养素建议。
- medical-markers：血脂、血糖、肝肾功能等体检指标解释。
- exercise：力量训练、有氧训练和恢复安排。
- supplements：常见补剂的适用场景和注意事项。
- medications：用药记录、频率和风险提示。

## 归档价值

该文档适合作为个人健康 Agent 的规则入口，也适合测试文档摘要和主题归类能力。
