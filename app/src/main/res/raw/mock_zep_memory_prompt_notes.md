# Zep Cloud 长记忆 Prompt 调优

目标：为多轮 Agent 工作流保留稳定偏好、关键事实和项目约束。

## 记忆边界

- session memory 记录当次任务状态。
- user facts 记录长期偏好、禁忌和固定环境。
- project graph 记录仓库约束、模块职责和未完成决策。

## Prompt 约束

1. 写入事实前先判断是否长期有效。
2. 不保存 API key、个人证件号或一次性验证码。
3. 检索时优先项目约束，再合并用户偏好。
4. 低置信事实进入 pending 区，等待二次确认。

## 自动化草图

输入消息 -> 事实抽取 -> 敏感信息过滤 -> Zep 写入 -> 下次任务检索注入。
